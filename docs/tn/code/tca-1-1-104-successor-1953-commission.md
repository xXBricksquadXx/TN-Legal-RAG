## Practitioner summary

This section ties the current Tennessee code commission back to the body created in Chapter 80 of the 1953 Public Acts.
The present commission is declared to be the immediate successor to that earlier commission.

## Key practice points

- Confirms continuity of the commission’s authority from the 1953 act to the modern statute.
- Useful when reading older contracts or references that point to the 1953 commission.
